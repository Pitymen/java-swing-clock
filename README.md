# java-swing-clock

Aplicação desktop simples em Java Swing que exibe data e hora no formato `dd/MM/yyyy HH:mm:ss`.

## Funcionalidades
- Exibe data/hora ao abrir a janela
- Botão para atualizar manualmente
- Interface fixa (layout absoluto) com ícone à esquerda e texto à direita

## Tecnologias
- Java (Swing)
- `java.time` (`LocalDateTime`, `DateTimeFormatter`)

## Como executar
1. Abra o projeto na IDE (IntelliJ, Eclipse, etc.)
2. Execute a classe `TelaRelogioMy`

## Estrutura de arquivos
- `src/` (código-fonte)
- `resources/imagens/calendario.png` (ícone usado na interface)
